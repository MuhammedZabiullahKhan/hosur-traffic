# 🚦 Hosur Traffic Control Center

## Created by Shri Muhammed Zabiullah Khan @Axion_Z_Squad

### 🌟 Features
- **Real-time Traffic Monitoring** using AQI data
- **6 Probability Methods**: Bayesian, PSI, PMF, CI, Ensemble, Location-Specific
- **Multi-Station Support**: Hosur + Bangalore stations
- **Physics-Based PSI** with temperature, pressure, humidity, wind adjustments
- **Beacon Traffic Light** with pulse animations
- **Cyberpunk Theme** with moving car animations
- **Auto-Refresh** every 60 seconds
